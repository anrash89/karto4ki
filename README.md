# Детский Обучающий Планшет — Веб-каталог карточек

Этот проект представляет собой интерактивную галерею для просмотра 225 обучающих детских карточек. Дизайн страниц адаптирован под внешний вид реальных физических карт (со скругленными цветными рамками, категориями, номерами и техническим штрихкодом слева).

## Функционал
1. **Фильтрация по 12 категориям** (Животные, Транспорт, Профессии, Формы и др.).
2. **Мгновенный живой поиск** по названию слова.
3. **Режим просмотра (Зум)**: при клике на карту открывается модальное окно крупным планом.
4. **Аудио-озвучка**: при открытии карточки или клике на кнопку «Слушать звук» воспроизводится аудиофайл.
5. **Умные заглушки**: если картинка или звук еще не загружены в папки проекта, карточка автоматически выведет красивый крупный эмодзи, соответствующий её тематике, сохранив интерфейс рабочим.

## Структура папок медиафайлов
Чтобы наполнить сайт вашими изображениями и звуками, сохраняйте их в корневую директорию проекта по следующим путям:

* **Изображения:** Папка `images/[имя_категории_на_латинице]/[слово_маленькими_буквами].png`
* **Звуки:** Папка `audio/[имя_категории_на_латинице]/[слово_маленькими_буквами].mp3`

### Соответствие папок (slugs):
* Животные `->` `images/animals/`
* Транспорт `->` `images/transport/`
* Предметы `->` `images/items/`
* Еда `->` `images/food/`
* Фрукты `->` `images/fruits/`
* Овощи `->` `images/vegetables/`
* Одежда `->` `images/clothes/`
* Природа `->` `images/nature/`
* Семья `->` `images/family/`
* Профессии `->` `images/professions/`
* Цвета `->` `images/colors/`
* Фигуры `->` `images/shapes/`

*Пример для карточки «Пожарная машина» в категории «Транспорт»:*
* Картинка: `images/transport/пожарная_машина.png`
* Звук: `audio/transport/пожарная_машина.mp3`

## Как запустить / Залить на GitHub Pages
1. Распакуйте архив.
2. Создайте новый репозиторий на вашем GitHub.
3. Загрузите файлы `index.html`, `styles.css`, `app.js` и `README.md` в ваш репозиторий.
4. Зайдите в **Settings** репозитория -> вкладка **Pages** -> выберите ветку `main` (или `master`), нажмите **Save**.
5. Через 1-2 минуты проект будет доступен по адресу `https://ваш-логин.github.io/имя-репозитория/`.
