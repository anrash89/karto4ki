# Детский Обучающие Карточки — Мобильная Версия

Проект интерактивного каталога (225 слов), полностью оптимизированный под мобильные устройства и планшеты.

## Что сделано для мобильных:
1. Карточки выводятся адаптивной сеткой по 2 штуки в ряд на смартфонах.
2. Категории сверху прокручиваются свайпом.
3. Отключены дефолтные мобильные меню браузеров на долгий тап по элементам.
4. Размеры всех кнопок оптимизированы для удобных кликов пальцами.

Разверните файлы на GitHub Pages согласно основной инструкции!
